SQL Server tables

The SQL Server 2000 database was chosen because it is a robust and secure application easily integrated with Visual Studio .Net applications software. In the project, 12 tables are used to record data. Please click on the table name below to see a representation of its contents. (The window which appears is resizeable)

active_issues records the top five active issues in the dow for each day
details shows a little background information on each dow component  
dow shows general details of the level of the DOW index for each day
dow_estimates   
hist_active_issues a table with results of calculations of historical most active issues in the dow
hist_shares historical shares, 200 days from April 20th 2003
history historical prices for the dow components
history_backup a backup of the historical prices  
multiple_regression_tally contains data derived from the share prices of a form compatible with Y = ß0 + ß1X1 + ß2X2 + ß3X3 + ß4X4 + ß5X5 + e
regression_tally contains data derived from the share prices of a form compatible with Y = ß0 + ß1X + e
shares contains records of the daily prices for each share in the dow
shares_outstanding contains the number of shares on the market for each dow component
